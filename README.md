# Nabu Casa, Inc. Website

The website is built with [11ty](https://www.11ty.dev/) and deployed on [Cloudflare](https://www.cloudflare.com/)

## URLs
All non-www urls are redirected to www

- Primary: https://www.nabucasa.com/
- Preview: https://preview.nabucasa-com.pages.dev/
- Branch: https://{branch}.nabucasa-com.pages.dev/

## Test

`npm run start`

## Deployment

Any change and PRs are against preview build. If the preview works fine, we can merge it into master.

Branch/URL:

- preview -> Preview URL
- master -> Primary URL
